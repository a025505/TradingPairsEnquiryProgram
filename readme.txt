## PLEASE INSTALL requests package IN YOUR PYTHON ENVIRONMENT TO RUN THE APPLICATION ##

This program will only show the trading pairs that you saved in the support_data.py file.

Add the trading pairs you would like to query in the dict_pairing dictionary (follow the convention).

For more information about the WhiteBit API, you may refer to: https://github.com/whitebit-exchange/api-docs

(FYI WhiteBit is not a popular trading platform, this file is just showing how I implement and call their public api in my program. Most trading platform have sample code 
that can help you get started, you may want to refer to those instead and implement it in the code.)
